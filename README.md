# date-spread
Create string data of days spread between 2 dates.

**HOW TO USE:**

Firstly add this directory to Environment Variable.<br/>
Then type: <br/>
```
date-spread --from "01/01/2024" --to "31/12/2024" --count "100" --out "file.txt"
```
>Expected output:
>
>01/01/2024<br/>
>06/02/2024<br/>
>14/03/2024<br/>
>19/04/2024<br/>
>26/05/2024<br/>
>01/07/2024<br/>
>07/08/2024<br/>
>13/09/2024<br/>
>19/10/2024<br/>
>31/12/2024<br/>
